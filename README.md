Control Center
The Control Center is a comprehensive system management tool designed to provide users with a wide range of functionalities to manage their system efficiently. Below is a list of the key features available in the Control Center:

Features
User Name Display: Shows the current logged-in user's name.
Date Display: Displays the current date.
Time Display: Shows the current time.
System Shutdown: Allows the user to shut down the system.
System Restart: Enables the user to restart the system.
Sleep Mode: Puts the system into sleep mode.
Settings Navigation: Directs the user to the system settings.
System Lock: Locks the system, requiring authentication to unlock.
Log Out: Logs out the current user.
Task Manager: Opens the task manager to monitor and manage running applications.
Close All Applications: Closes all currently running applications.
Calculator: Provides access to a built-in calculator.
Status Display: Shows various system statuses (e.g., battery percentage, network status).
Volume Control: Allows adjustment of the system's audio volume.
Monitor Brightness Control: Enables the user to adjust the brightness of the monitor.

Usage
To use the Control Center, simply navigate through the options available in the interface. Each feature is designed to be intuitive and user-friendly, ensuring that you can manage your system with ease.
